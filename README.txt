README file:
------------

The time series is predicted using ARIMA model using R language.

## Environment:
Running R on Windows,
Downloaded Download R 3.4.0 for Windows from https://cran.r-project.org/bin/windows/base/ 
R-3.4.0\bin\x64 (64-bit) was downloaded, Rgui.exe runs as a 
standard Windows GUI executable and provides an R console in its own window.

forecast package (forecast_8.0) is installed onto the evnironment.

----------------------------------------------------------------------
## To compile and run: 
Assuming you are in the directory containing this README:

The folder named shali_saseendran_project contains product_distribution_training_set.csv
and key_product_IDs.csv file for the processsing of the code.

product_distribution_training_set.csv -> column (product id) and row (days)

Unzip the folder and extract the files.

Load the timeseriesforecasting.r file to the R language environment and 
run the code.
Select the whole file and right click, select run.

OR

Copy/paste the whole file data into the R window console.
